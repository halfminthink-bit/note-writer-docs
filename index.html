note & WordPress Auto Publisher

Google Apps Scriptで実現する、AI記事自動生成・投稿システム

主な機能

AI記事生成

OpenAI APIを使用して、キーワードから高品質な記事を自動生成

Google Docs保存

生成された記事を自動的にGoogle Docsに保存

note.com投稿

ブラウザ自動操作でnoteに記事を自動投稿

WordPress投稿

WordPress REST APIで記事を自動投稿

スプレッドシート管理

Google Sheetsで記事の生成・投稿を一元管理

バックグラウンド処理

Google Cloud Runで高速・安定した処理を実現

使い方

  1. スプレッドシートにキーワードを入力
    記事生成したいトピックのキーワードを設定
  2. メニューから記事生成を実行
    AIが自動的に記事を生成してGoogle Docsに保存
  3. noteまたはWordPressに投稿
    ワンクリックで記事を自動投稿
  4. 投稿結果を確認
    スプレッドシートに投稿URLが自動記録

セキュリティとプライバシー

本サービスは、Googleの厳格なセキュリティ基準を満たしています：

  • Google Cloud Platformの認証システムを使用
  • すべてのAPI通信は暗号化(HTTPS/TLS)
  • 最小権限の原則に基づくアクセス制御
  • ユーザーデータは全てGoogle Drive内に保存
  • 第三者とのデータ共有は最小限(OpenAI APIのみ)
