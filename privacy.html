プライバシーポリシー

最終更新日: 2025年11月2日

1. はじめに

note & WordPress Auto Publisher（以下「本サービス」）は、Google Apps Scriptを使用して、 記事の自動生成および投稿を支援するサービスです。本プライバシーポリシーは、 本サービスがどのようにユーザーの情報を収集、使用、保護するかを説明します。

2. 収集する情報

本サービスは以下の情報にアクセスします：

  • Google Spreadsheetsデータ: 記事生成のためのキーワード、投稿先情報
  • Google Docsデータ: 生成された記事コンテンツ
  • Google Drive: ドキュメントの作成と共有のため
  • Cloud Tasks API: バックグラウンド処理のため

3. 情報の使用目的

収集した情報は以下の目的でのみ使用されます：

  • OpenAI APIを使用した記事の自動生成
  • 生成された記事のGoogle Docsへの保存
  • note.comおよびWordPressへの記事投稿
  • 投稿状況のスプレッドシートへの記録

4. 第三者との情報共有

本サービスは以下の第三者サービスと情報を共有します：

  • OpenAI API: 記事生成のためにキーワードと生成指示を送信
  • Google Cloud Platform: サービスの実行とデータ処理のため
  • note.com: 記事投稿のため
  • WordPress: 記事投稿のため

上記以外の第三者とユーザー情報を共有することはありません。

5. データの保存場所

ユーザーのデータは以下の場所に保存されます：

  • Google Drive: ユーザーのGoogle Driveアカウント内
  • Google Cloud Platform: 東京リージョン（asia-northeast1）

本サービスは独自のデータベースにユーザー情報を保存しません。

6. データのセキュリティ

本サービスは以下のセキュリティ対策を実施しています：

  • Google Cloud Platformの認証システムを使用
  • API通信の暗号化（HTTPS/TLS）
  • 最小権限の原則に基づくアクセス制御
  • サービスアカウントによる安全な認証

7. ユーザーの権利

ユーザーは以下の権利を有します：

  • アクセス権: 自身のデータにアクセスする権利
  • 削除権: データを削除する権利
  • 取り消し権: サービスへのアクセス権限を取り消す権利

8. データの削除方法

ユーザーは以下の方法でデータを削除できます：

  • Google Apps Scriptのアドオンをアンインストール
  • Google SpreadsheetsおよびGoogle Docsを削除
  • Google Cloud Consoleで本サービスへのアクセス権限を取り消し

9. 子どものプライバシー

本サービスは13歳未満の子どもを対象としていません。 13歳未満の子どもから故意に個人情報を収集することはありません。

10. プライバシーポリシーの変更

本プライバシーポリシーは必要に応じて更新されることがあります。 重要な変更がある場合は、本ページに掲載してお知らせします。

Privacy Policy (English)

Last Updated: November 2, 2025

1. Introduction

note & WordPress Auto Publisher ("the Service") is a Google Apps Script-based service that helps automate article generation and publishing. This Privacy Policy explains how the Service collects, uses, and protects user information.

2. Information We Collect

The Service accesses the following information:

  • Google Sheets Data: Keywords for article generation, publishing destinations
  • Google Docs Data: Generated article content
  • Google Drive: For document creation and sharing
  • Cloud Tasks API: For background processing

3. How We Use Information

Collected information is used only for the following purposes:

  • Automated article generation using OpenAI API
  • Saving generated articles to Google Docs
  • Publishing articles to note.com and WordPress
  • Recording publishing status in spreadsheets

4. Information Sharing with Third Parties

The Service shares information with the following third-party services:

  • OpenAI API: Sends keywords and generation instructions for article creation
  • Google Cloud Platform: For service execution and data processing
  • note.com: For article publishing
  • WordPress: For article publishing

We do not share user information with any other third parties.

5. Data Storage

User data is stored in the following locations:

  • Google Drive: Within the user's Google Drive account
  • Google Cloud Platform: Tokyo region (asia-northeast1)

The Service does not store user information in its own database.

6. Data Security

The Service implements the following security measures:

  • Uses Google Cloud Platform authentication system
  • Encrypted API communications (HTTPS/TLS)
  • Access control based on the principle of least privilege
  • Secure authentication using service accounts

7. User Rights

Users have the following rights:

  • Access Right: Right to access their own data
  • Deletion Right: Right to delete their data
  • Revocation Right: Right to revoke service access permissions

8. How to Delete Data

Users can delete their data by:

  • Uninstalling the Google Apps Script addon
  • Deleting their Google Sheets and Google Docs
  • Revoking service access permissions in Google Cloud Console

9. Children's Privacy

The Service is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.
